# Global-Food-Security-And-Agricultural-Market-Analytics

1. Objective
The objective of this project is to analyze the relationship between commodity prices, fertilizer prices, crop production, and food security indicators in order to identify trends, risks, and possible food security challenges over time.
The dashboard helps answer questions such as:
Are food prices increasing?
Are fertilizer costs affecting agriculture?
Is crop production improving or declining?
Are crisis and supply stress situations rising?
How are agricultural inputs linked to food security outcomes?

Aim of the Project

The aim of this project is to build an interactive Power BI dashboard that gives a complete view of:
commodity price movement,
agricultural productivity,
food affordability,
food crisis indicators,
and long-term food security risk patterns.
This project supports decision-making for analysts, policymakers, and business stakeholders by turning raw data into meaningful insights.

Project Conclusion

how food and fertilizer prices change over time,
how crop production responds to these changes,
how undernourishment and food shock risk behave,
and how crisis and supply stress indicators reveal food security pressure.
